Requerimientos Técnicos 

 

Para experiencias web móviles

  • Navegador Chrome.
  • Dispositivos Android (posiblemente iOS) smartphones y tablets.
  • Los archivos correspondientes a cada elemento interactivo se almacenará en la Plataforma Convida (web) y recibirán ciertos datos de ésta, tales como el id del usuario, su nombre, y algunos otros datos de persistencia que se requieran, tales como el nivel de dificultad seleccionado, el número de interacciones previas, etc.
  • Se intentará hacer uso de las APIs de almacenamiento local, etc., etc. para ofrecer una experiencia offline al usuario cuando sea factible, a manera de las Progressive Web Apps.
