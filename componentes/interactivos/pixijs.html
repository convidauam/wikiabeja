Además de los videojuegos desarrollados en Unity, el proyecto Convida contempla el desarrollo de juegos o actividades más sencillas que serán creados usando el grafo de escena PixiJS.

Los juegos o tipos de actividades contemplados hasta ahora son los siguientes:

  • Combinabeja (Tragamonedas generador de avatar):  Mini-juego interactivo donde el usuario genera un avatar único y aleatorio. El sistema utiliza un mecanismo de tragamonedas basado en diversas especies de abejas dando como resultado un avatar unico con estilo pixel art

 

Arquitectura para la generación de los interactivos PixiJS

 

Caracteristicas

  • Creación del combina mediante la configuración de un formulario externo
  • Juego principal para generar el avatar del usuario con slots giratorios y palanca
  • Ajustador de avatar para arrastrar, redimensionar o resetear en caso de ser necesario
  • Exportación de un zip para tener el combinador con los ajustes finales
  • Compatible en version movil y de escritorio

Arquitectura del combinador

Para el desarrollo del “CombinaTodo"  se ha implementado una estructura basada en PixiJS, organizada de la siguiente manera:

├── src/

│   ├── core/  

│   │   └── game.js                                # Controlador principal

│   │   └── CombinaGenerator.js         # Orquestador

│   │   └── CombinaUI.js                        # Interfaz visual

│   │   └── CombinaSpin.js                     #Logica de los giros

│   │   └── CombinaExport.js                 # Exportador

│   │   └── ConfigManager.js                 # Guarda y carga la configuracion

│   │   └── AssetLoader.js                       # Carga asincrona imagenes

│   ├── utils/

│   │   ├── imageLoader.js                     # Cargador optimizado de  imagenes   

│   │   └── seededRandom.js                # Generador de aleatorio

│   └── views/

│       ├── AvatarAdjuster.js                       # Editor de avatar

│       ├── DynamicSlotViewr.js                 # Vista de los slots

│       └── uiLever.js                                     # Palanca

├── main.js                          # Punto de entrada

├── style.css                       # Estilos globales

├── index.html

 

 

Flujo de creacion del combinador

  1. Abrir y completar el formulario /form-standalone.html
  1. Generar el JSON con la configuración
  1. Abrir aplicacion e ingresar el JSON
  1. Probar generador y ajustar avatar de ser necesario
  1. Exportar paquete y descomprimir ZIP
  2. Abrir index.html y disfrutar combinador en version movil o de escritorio
