Se planea la creación de una interfaz gráfica en un entorno web interactivo en donde se puedan generar representaciones gráficas de ciertos contenidos teóricos/temáticos por medio del uso de un lenguaje gráfico que denote ciertas relaciones entre conjuntos y conceptos de un cierto campo de saberes.

Las representaciones generadas de esta manera permitirán, una vez que sean publicadas, a quien acceda interactuar con versiones gamificadas de esas representaciones visuales, en las cuales se generen mecánicas de construcción/reconstrucción parcial o desde cero, a partir de ciertas ayudas (o hints).

La interfaz gráfica en la que se generen estas representaciones gráficas forma parte del editor gráfico del Editor CONVIDA.
