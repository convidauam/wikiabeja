Plataforma Convida

La plataforma Convida es un sitio web dinámico diseñado para albergar contenidos temáticos, elementos interactivos y mapas conceptuales/grafos relacionados con la estructura teórica de cada Contexto Virtual de Aprendizaje (CONVIDA).

Estos contenidos, así como los datos de las personas usuarias se almacenarán en una base de datos basada en la persistencia de objetos, y los medios asociados a cada contenido (imágenes, videos, scripts, etc.) se almacenarán de manera local, a través de un servidor web, o de manera alternativa en algún servicio de almacenamiento en la nube. En ambos casos, la Plataforma Convida es responsable de controlar el acceso a esos medios, así como de establecer formas en las que las personas administradoras, docentes, programadores, etc. publican o actualizan esos recursos.

De igual manera, la plataforma llevará un registro de todas las interacciones de las personas usuarias, particularmente de aquellas acciones efectuadas por los y las estudiantes, y las cuales se registrarán, se actualizarán y se consultarán por medio de un API compatible con el estándar ActivityPub.
